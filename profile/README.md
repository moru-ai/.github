## About Moru

Moru provides secure sandbox infrastructure for AI agents. Each sandbox runs in a hardware-isolated Firecracker microVM with full filesystem access, command execution, and real-time streaming.

<h3>Getting Started</h3>

- Start with the [Quickstart guide](https://moru.io/docs/quickstart)

<h3>Repositories</h3>

- [Moru](https://github.com/moru-ai/moru): Python SDK (`moru`) and JavaScript SDK (`@moru-ai/core`) for sandbox control
- [Sandbox Infrastructure](https://github.com/moru-ai/sandbox-infra): Open-source Firecracker-based sandbox infrastructure
- [Maru](https://github.com/moru-ai/maru): A research assistant built with Moru sandbox and [Claude Agent SDK](https://platform.claude.com/docs/en/agent-sdk/overview)

<h3>Examples</h3>

- [Maru](https://github.com/moru-ai/maru): A web clone of Claude Code built with Moru sandboxes and Claude Agent SDK.

Visit [moru.io/docs](https://moru.io/docs) for documentation.
